##//1.biology.cpp--生物种群密度实验模拟
概述:这是一个计算机模拟实验，用于比较四种生物种群密度调查方法的准确性。模拟在一个1平方米的区域内进行，初始种群约为1000个生物个体。
核心内容
1. 模拟设置·
   区域：1平方米正方形·
   初始种群：900-1100个（随机）·
   模拟时长：30天·
   个体特征：位置、年龄、移动速度·
   环境限制：最大密度5000个/平方米
2. 种群动态·
   年龄增长：每天增加1天·
   年龄分段：幼年（0-180天）、成年（181-540天）、老年（541-730天）·
   死亡：年龄相关死亡率（幼年0.3%、成年0.1%、老年0.5%）·
   繁殖：成年个体繁殖，后代在父母附近出生·
   迁移：迁入和迁出个体·
   移动：每天随机移动
3. 调查方法对比
   逐个计数法· 直接计数所有个体·
   .优点：100%准确·
   .缺点：不适用于大面积或活动性强的种群
   五点样方法· 在5个固定位置取样·
   .优点：快速简便·
   .缺点：可能错过聚集区域
   等距样方法· 沿直线等距取10个样方·
   .优点：代表性较好·
   .缺点：可能不适合特殊分布
   标记重捕法· 捕捉-标记-释放-重捕·
   .优点：适用于活动性种群·
   .缺点：计算结果不稳定
4. 生态学原理·
   密度制约：种群密度影响出生率和死亡率·
   年龄结构：不同年龄个体对种群贡献不同·
   空间分布：个体移动影响分布和调查结果
5. 实验结果·
   逐个计数法始终准确·
   样方法通常有3-10%的误差·
   标记重捕法误差波动较大（2-50%）·
   种群密度随时间自然波动
6. 实际意义· 展示了不同调查方法的特点· 帮助理解生态学中的抽样误差· 为实际调查提供方法选择参考结论这个实验展示了生态学中常用的四种种群密度调查方法，通过计算机模拟对比了它们的准确性和适用性。实验结果说明，没有一种方法是完美的，实际应用中需要根据种群特性和研究目的选择合适的方法。
